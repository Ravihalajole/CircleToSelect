##  What is Circle To Select?

**Circle To Select** is a fork of [CircleToSearch](https://github.com/AKS-Labs/CircleToSearch), allows you to select any part of your screen to extract text, or scan QR codes—all while keeping your data **100% on-device**. 

Built for privacy enthusiasts and power users, it works on any Android 10+ device, including De-Googled ROMs, without requiring an internet permission.

---

##  Core Features

###  Intelligent OCR & Extraction
- **Bundled ML Kit OCR**: High-speed, high-accuracy text recognition that works 100% offline. No more manual Tesseract model management.
- **Text-First Workflow**: Optimized for efficiency—when a gesture is triggered, the app immediately launches the OCR selection layer as its primary objective, allowing you to interact with on-screen text as if it were a live document.
